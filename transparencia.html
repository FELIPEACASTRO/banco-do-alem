O nosso juramento

  • Nenhum lucro, para ninguém. Nem para o criador, nem para investidor. Este projeto não existe para ganhar dinheiro — existe para lembrar que ele não é tudo.
  • Cada centavo, à vista. Toda contribuição e toda despesa aparecem aqui, com data. Sem letra miúda.
  • Só o custo, nada além. As doações cobrem apenas o que mantém a obra de pé (hoje, isso é praticamente zero). O que sobrar vai, integralmente, para uma causa votada pela comunidade — cuidados paliativos, moradores de rua, quem precisa.
  • Nunca vendemos você. Nada que você digita sai do seu aparelho. Não há cadastro, não há cookie de rastreio, não há venda de dados. A diferença entre nós e um banco de verdade é exatamente essa.
  • Código aberto. Qualquer pessoa pode ver como isto foi feito — e usar para criar a sua própria versão.

"Você não é dono da fortuna. É apenas seu depositário — e dela prestará contas." Se acreditamos nisso, temos de viver isso. Este cofre de vidro é a nossa prestação de contas.

A confiança é o único patrimônio que um banco de verdade não consegue comprar.
