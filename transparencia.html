IIO Juramento

Cinco compromissos

O nosso juramento

Não são termos de uso. São promessas que, se descumpridas, esvaziam a razão de este projeto existir.

  1. I

    Nenhum lucro, para ninguém

    Nem para o criador, nem para investidor. Este projeto não existe para ganhar dinheiro — existe para lembrar que ele não é tudo.

  2. II

    Cada centavo, à vista

    Toda contribuição e toda despesa aparecem aqui, com data. Sem letra miúda, sem rodapé que desmente o título.

  3. III

    Só o custo, nada além

    As doações cobrem apenas o que mantém a obra viva. O que sobrar vai, integralmente, para uma causa votada pela comunidade.

  4. IV

    Nunca vendemos você

    Nada que você digita sai do seu aparelho. Sem cadastro, sem rastreio, sem venda de dados.

  5. V

    Código aberto

    Qualquer pessoa pode ver como isto foi feito — e criar a sua própria versão.
